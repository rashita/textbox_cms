# 記法サンプル

このページでは使える記法を確認できます。

## ページ間リンク

`[[ページ名]]` と書くと内部リンクになります。

例: [[index]] でトップページに戻ります。

## 見出し

```
# 見出し1
## 見出し2
### 見出し3
```

## テキスト装飾

**太字** / *斜体* / ~~取り消し線~~

## リスト

- 項目A
- 項目B
  - ネスト項目

1. 番号付きリスト
2. 二番目

## 引用

> 引用はこのように表示されます。
> 複数行も可能です。

## コード

インラインコード: `const x = 1`

ブロックコード:

```javascript
function hello() {
  console.log("Hello, World!");
}
```
